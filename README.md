# Open The Doc README

#### 😋 配置链接

配置链接的方式有 2 种，如下：

1. 设置 > 扩展 > Open The Doc > 在 settings.json 中编辑 > 加入自定义链接键值对
2. 点击左侧图标 > 点击+号 > 按照步骤填写链接

#### 😋 打开文档

打开链接的方式有 3 种，如下：

1. 从左侧图标进入链接列表，点击链接即可打开。
2. 按 F1 打开命令面板，搜索“open the doc”，选择“😋 Open The Docs Picker”打开文档选项，选择文档打开。
3. 这种方式适用于打开临时链接。在文件内选中要打开的链接，鼠标右键选择“😋 Open In VS Code”。
